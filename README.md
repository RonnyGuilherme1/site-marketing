# site-marketing (Static Site)

Este projeto e o **site de apresentacao** (Static Site no Render).

## Como publicar no Render

- Tipo: Static Site
- Build command: (vazio)
- Publish directory: `.`

## Ajustar dominio do app

Edite `assets/js/config.js` e troque `https://app.seudominio.com` pelo dominio real do seu Web Service.

Exemplo:

```js
window.APP_BASE_URL = 'https://app.seu-dominio.com';
```
